Perguntas Frequentes

Respostas rápidas sobre instalação, uso e segurança

O aplicativo coleta dados?

Privacidade e armazenamento local

Não. O app funciona offline e não envia informações para servidores. Todos os dados ficam salvos localmente em %APPDATA%/DahoraApp com controle total do usuário.

Funciona sem Python?

Requisitos e versões disponíveis

Sim. Use a versão .exe compilada que não requer instalação. A versão script exige Python 3.8+ e dependências do arquivo requirements.txt.

Qual é o atalho global?

Uso e personalização de atalhos

O atalho padrão é Ctrl+Shift+Q. Funciona em qualquer aplicativo Windows e pode ser personalizado no arquivo de configuração se necessário.

Posso personalizar o formato?

Padrão e planos futuros

O formato atual é fixo [DD.MM.AAAA-HH:MM] para manter consistência. Futuras versões podem oferecer opções de personalização baseadas em feedback da comunidade.

Onde ficam salvos os dados?

Armazenamento e backup

Os dados ficam em %APPDATA%/DahoraApp. Incluem histórico, configurações e logs. Para backup, copie esta pasta ou use "Limpar Histórico" no menu.

Como resolver problemas?

Troubleshooting e suporte

Verifique o arquivo qopas.log na pasta de dados. Use "Limpar Histórico" no menu ou reinicie o aplicativo para resolver a maioria dos problemas.

Privacidade & Dados

Transparência total sobre armazenamento local e segurança

Armazenamento Local

Histórico de clipboard armazenado exclusivamente em %APPDATA%/DahoraApp. Dados nunca saem do seu computador.

Logs de Diagnóstico

Arquivos qopas.log criados localmente para troubleshooting. Sem envio de informações externas.

Zero Telemetria

Sem rastreamento, analytics, anúncios ou coleta de dados. Aplicativo funciona completamente offline.

Controle Total

Você tem controle completo. Use "Limpar Histórico" no menu para apagar todos os dados quando quiser.

Backup Simples

Basta copiar a pasta %APPDATA%/DahoraApp para backup completo de histórico e configurações.

Código Aberto

Transparência total. Você pode inspecionar o código-fonte e verificar por si só a segurança implementada.
