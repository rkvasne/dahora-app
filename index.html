Perguntas Frequentes

Respostas rápidas sobre instalação, uso e segurança

O aplicativo coleta algum dado?

Não. O Dahora App não envia informações para servidores. Histórico e configurações ficam salvos localmente.

Funciona sem Python instalado?

Sim, ao usar a versão .exe (quando disponível). A versão script exige Python 3.8+.

Qual é o atalho global?

Ctrl+Shift+Q copia a data/hora de qualquer aplicativo.

Posso personalizar o formato?

O formato atual é fixo para padronização. Futuras versões podem oferecer opções de personalização.

Privacidade & Dados

Transparência total sobre armazenamento local e segurança

  • Histórico de clipboard armazenado em %APPDATA%/DahoraApp.
  • Logs para diagnóstico em qopas.log (somente local).
  • Sem telemetria, anúncios ou rastreamento.
  • Você controla os dados — opção "Limpar Histórico" no menu.
